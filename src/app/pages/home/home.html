Tecnologías principales

CreativIA está construido sobre los siguientes frameworks, herramientas y APIs.

Angular

Framework de desarrollo frontend en el que se basa la arquitectura de la aplicación web. Gestiona aspectos como la navegación y la interacción con el usuario.

Firebase

Plataforma backend en la nube empleada para la gestión de datos (almacenamiento de paletas, sincronización en vivo...).

Bootstrap

Framework CSS que facilita el diseño web responsive a través de una gama de componentes y estilos para la creación de interfaces.

The Color API & Colormind

APIs especializadas en análisis y generación cromática. Proporcionan esquemas de color, convierten valores y asisten en la creación de paletas basadas en conceptos visuales.

GitHub

Plataforma de control de versiones del código fuente, documentación del proyecto y colaboración. Ofrece trazabilidad de cambios y publicación abierta.

Visual Studio Code

Entorno de desarrollo utilizado para la implementación del proyecto. Ofrece soporte y depuración integrada al resto de tecnologías que componen CreativIA.
